Design

It is easy to get lost when making decisions about the flow of a user interface or the layout of a print asset. Ideas can come from anywhere and knowing which ideas to develop for a particular user group requires focus.

That’s why I always focus on the goals, mode, and environment of the end user or viewer. It simplifies the decision making process and lets us put more of our creative energies into making remarkable designs.

Cross-Functionality

Every perfect solution has an expiration date. Users change. The digital ecosystem changes. The real ecosystem changes. Holding too tightly to any particular solution shifts focus away from the reason for the solution: the user.

I am always asking myself, “How can we help this person?” Sometimes, the answer is removing a feature. Other times it is writing code that a remote developer can easily read. Whatever the case, the benefit of the user is always the focus.
