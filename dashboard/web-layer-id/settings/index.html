Data Settings

Data is Editable By:

Editing data changes the data set of the layer permanantly. If the layer is a view, reference, or analysis layer, the data may not be stored in the layer at all. If this is the case the data for the layer must be edited at the source layer.

Track Edits:

Tracking edits presumably provides the layers owner with a record of what has happened to the data in the layer.

Data Sync:

This layers data is a reference from a source layer. Enabling sync will autmatically update this layers data any time the data at the source layer is updated.

Export:
Update
